# 🤝 Katkıda Bulunma Rehberi / Contributing Guide

## 🇬🇧 English
We welcome community engagement primarily through bug reporting, triaging, and feature conceptualization. 
- To report a defect, please utilize our formal [Bug Report Template](https://github.com/Emre37destan/APK-IDE-PRO-2026/issues/new?template=bug_report.yml).
- To request enhancements, please fill out the [Feature Request Template](https://github.com/Emre37destan/APK-IDE-PRO-2026/issues/new?template=feature_request.yml).

*Note: Since the core codebase remains a proprietary distribution model, merge requests affecting source mechanics cannot be processed.*

---

## 🇹🇷 Türkçe
Topluluk katılımını temel olarak hata bildirimleri, triyaj yönetimi ve yeni özellik fikirleri üzerinden memnuniyetle kabul ediyoruz.
- Bir kararsızlık veya hata bildirmek için [Hata Bildirim Şablonunu](https://github.com/Emre37destan/APK-IDE-PRO-2026/issues/new?template=bug_report.yml) kullanın.
- Yeni bir kabiliyet veya geliştirme önermek için [Özellik İstek Şablonunu](https://github.com/Emre37destan/APK-IDE-PRO-2026/issues/new?template=feature_request.yml) doldurun.

*Not: Çekirdek kaynak kod yapısı kapalı kaynak dağıtım modeline tabi olduğundan, kaynak kod mekanizmasını doğrudan etkileyen Pull Request (PR) birleştirmeleri işleme alınamayacaktır.*
