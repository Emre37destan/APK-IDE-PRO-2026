# 📚 Installation Guide / Kurulum Rehberi

## 🇬🇧 English
### Prerequisites
2. **Operating System:** Windows 10 or Windows 11 (64-bit).

### Installation Steps
1. Navigate to the [Releases](https://github.com/Emre37destan/APK-IDE-PRO-2026/releases) page.
2. Download `APK-IDE-PRO-2026.exe`.
3. Double-click the installer and follow the on-screen wizard instructions.
4. Launch **APK-MASTER 2026 PRO** using the desktop shortcut.
