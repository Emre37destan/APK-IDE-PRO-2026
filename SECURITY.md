# 🔒 Güvenlik Politikası / Security Policy

If you discover a potential vulnerability within the application runtime environments, please **do not** disclose it publicly via public issues. 

Instead, please communicate directly with the maintainer to allow swift resolution.
**Contact Email:** `destan3673@gmail.com`

We take all security reports seriously and will respond within 48 hours.
